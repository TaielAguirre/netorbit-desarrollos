# NetOrbit Typography

## Font Family
**Sans-Serif:** `Inter`, `system-ui`, `sans-serif`

The application primarily uses **Inter** for a clean, modern, and highly readable interface.

## Usage Guidelines

### Headings
- **Font Weight:** Bold (700) or Semi-Bold (600)
- **Color:** `dark-900` (#0f172a)

### Body Text
- **Font Weight:** Regular (400)
- **Color:** `dark-600` (#475569) or `dark-700` (#334155)

### Interactive Elements (Buttons, Links)
- **Buttons (Primary):** Semi-Bold (600)
- **Links:** Medium (500)

## Tailwind Configuration Reference
```javascript
fontFamily: {
  sans: ['Inter', 'system-ui', 'sans-serif'],
},
```
